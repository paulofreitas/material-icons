Using this icon font

The easiest way to set up icon fonts for use in any web page is through Google Fonts. All you need to do is include a single line of HTML: 

    <link href="https://fonts.googleapis.com/icon?family=Material+Icons+Round" rel="stylesheet">

Similar to other Google fonts, the correct CSS will be served to activate the Material Icons Round font family specific to the browser.

An additional CSS class will be declared called .material-icons-round. Any element that uses this class will have the correct CSS to render these icons from the web font.
